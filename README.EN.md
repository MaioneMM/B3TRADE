# B3TRADE

At B3TRADE, you have access to real-time stock data on the Brazilian stock market and cryptocurrencies. We provide a platform that shows all the necessary data for you to track the Brazilian financial market.

Works with common Brazilian stock shares ending in 3 and 4. It also works with Investment Fund stocks ending with 11.

We also support cryptocurrencies and currency conversion.

## Features

### Real Time Data
We provide data from the Brazilian stock market in real time.

### Cryptocurrencies
Search for crypto data in any currency.

### Currencies
Search and convert multiple currencies.
